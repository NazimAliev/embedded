бистатическая секция
Принимает данные по UDP с zboard по одному каналу
Посылает данные по UDP на адрес и порт, указанные в командной строке (на multi bb)

Инициализация UDP в app.c
Данные принимаются по UDP в inBufSig, inBufRef в processLoop
Функция кодека UNIVERSAL_process вызывается в функции потока proc
Потоки чередуются на четный и нечетный, отправка UDP идет в каждом из двух потоков
По UDP посылаются массивы outBufCfar[ABUFSIZE/BYTESIZE], outBufAf[2*ABUFSIZE], размер в байтах

Благодаря двум потокам один поток может принять UDP и запустить DSP когда во втором обработка еще не завршена
